# 健康仪表盘 - 项目说明

> 个人健康数据管理系统，仅供个人使用

## 1. 项目背景

### 1.1 项目目标
- 整合 Apple Watch 健康数据
- 记录每日饮食和运动
- 可视化展示健康趋势
- 获得个性化健康建议

### 1.2 使用场景
- 通过飞书私聊记录每日健康数据
- 查看网页仪表盘了解健康趋势
- 追踪体重、运动、饮食变化

## 2. 功能需求

### 2.1 核心功能

| 模块 | 功能 | 说明 |
|:---|:---|:---|
| 数据收集 | Apple Health 同步 | 自动同步步数、心率、睡眠 |
| 数据收集 | 饮食记录 | 拍照记录每餐食物 |
| 数据展示 | 健康仪表盘 | 可视化展示健康趋势 |
| 数据分析 | 健康建议 | 基于数据提供建议 |

### 2.2 优先级

- **P0**: 用户管理、Apple Health 同步、饮食记录、基础仪表盘
- **P1**: 健康趋势分析、个性化建议
- **P2**: 高级分析、数据导出

## 3. 技术栈

| 层级 | 技术 | 说明 |
|:---|:---|:---|
| 前端 | React + TypeScript + Tailwind CSS | 用户界面 |
| 后端 | Python + FastAPI | API 服务 |
| 数据库 | PostgreSQL + Redis | 数据存储 |
| 部署 | Docker + Nginx | 容器化部署 |
| 监控 | Prometheus + Grafana | 系统监控 |
