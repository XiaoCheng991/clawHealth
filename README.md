# 健康仪表盘 (Health Dashboard)

## 项目介绍

个人健康数据管理系统，包括：
- Apple Watch 健康数据接入
- 食物图像识别与卡路里分析
- 健康数据可视化仪表盘
- 个人健康档案管理

## 开发流程

1. **需求分析**
2. **BRD (Business Requirements Document)**
3. **PRD (Product Requirements Document)**
4. **架构设计**
5. **详细设计**
6. **项目框架搭建**
7. **代码开发**
8. **测试**
9. **本地验收**
10. **部署上线**

## 技术栈

- **后端**: Python FastAPI
- **前端**: React + Tailwind CSS
- **数据库**: PostgreSQL
- **部署**: Docker + Nginx
